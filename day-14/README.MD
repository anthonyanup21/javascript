# ğŸ” JavaScript Array Methods â€” `some()` & `every()`  
### (Beginner-Friendly README)

Both `some()` and `every()` are array methods used to check **conditions** on array elements.  
They return **true** or **false** based on the condition.

---

# ğŸŸ¦ 1. `some()`  
### âœ” Purpose: Check if **at least one element** satisfies the condition  
### âœ” Returns: `true` / `false`

> **If even ONE element matches â†’ returns true**

#### Syntax:
```js

```

---

## âœ… Example 1: Check if array has even numbers
